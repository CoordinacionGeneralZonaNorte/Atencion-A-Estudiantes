Para subir la página que se está desarrollando, se necesita lo siguiente:
*Servidor
*Usuario
*Contraseña

¿Qué se pretende?
Hablar con los encargados de "los que suben las páginas web", que se denominan "Administradores de host" para que se pueda crear un espacio especial para la Coordinación de Atención A Estudiantes y Gestión Escolar de la Zona Norte.
El administrador del Host (un host es un equipo que está conectado a la red y que provee el servicio de almacenar un sitio web, es el lugar donde viven las páginas web) maneja el domino (servidor), por lo que el Administrador del Host crea y comparte la Cuenta y Contraseña por FTP
Por ejemplo: En el Tecnológico, se maneja lo siguiente:
*Servidor (Padre): www.itiguala.edu.mx (dominio)
*Usuario (Hijo): dsc.itiguala.edu.mx
*Contraseña: ************

Para la creación de la página web de la Coordinación de Atención a Estudiantes, se establece lo siguiente:
*Servidor (padre): www.uagro.mx
*Usuario (Hijo): CoordinacionAtencionEstudiantes.uagro.mx
*Contraseña: aEstudiantes2021
